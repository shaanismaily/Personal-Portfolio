A soft, creative personal portfolio website built with vanilla HTML, CSS, and JavaScript. No frameworks, no build tools — just clean, hand-crafted code.

## Features

- Smooth scroll navigation
- Animated floating blobs in the hero section
- Fade-in animations on scroll (Intersection Observer API)
- Fully responsive for mobile and desktop
- Sections: Hero · About · Skills · Projects · Contact

No installation or build step needed.

```bash

git clone https://github.com/your-username/portfolio.git

# Open in browser
open index.html
```

Or just drag `index.html` into your browser.

## Customization

All content is in `index.html`. Here's what to update:

- **Your name / tagline** — Hero section
- **About text** — About section
- **Projects** — Swap the 3 placeholder cards with your real projects
- **Contact links** — Update GitHub, LinkedIn, and email links
- **Location** — Currently set to Jhansi, Uttar Pradesh 🇮🇳

---

## Deployment

This is a single HTML file, so it deploys anywhere:

**GitHub Pages**
1. Push to a repo named `your-username.github.io`
2. Go to Settings → Pages → set source to `main` branch
3. Your site is live at `https://your-username.github.io`

**Netlify (drag & drop)**
1. Go to [netlify.com](https://netlify.com)
2. Drag your `index.html` into the deploy box
3. Done — live in seconds
